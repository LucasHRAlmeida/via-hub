# Bem-Estar Multissensorial

Guia web interativo que sugere combinações de **música, cor e aroma** para seis estados de espírito — Relaxamento, Foco e Produtividade, Alegria e Motivação, Introspecção e Reflexão, Energia e Vitalidade e Sono Profundo. Cada estado abre uma ficha com cromoterapia, aromaterapia, playlist sugerida (busca no Spotify), efeito combinado, referências e modo de uso.

## Acesso

- Página publicada: <https://lucashralmeida.github.io/via-hub/bem-estar-multissensorial/>

## Escopo e limitações

- Intervenções **inócuas e de baixo custo**, voltadas ao bem-estar geral.
- As referências citadas contextualizam as escolhas; a página **não substitui orientação profissional** nem alega efeito terapêutico.
- Os links de playlist apontam para a busca pública do Spotify, sem integração ou coleta de dados.

## Stack

Página única em **HTML5 + CSS3 + JavaScript** (vanilla, sem dependências externas), responsiva e com suporte a tema claro/escuro via `prefers-color-scheme`. Basta abrir o `index.html`.
